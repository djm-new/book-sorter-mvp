# Book Sorter MVP

A tiny private Railway app for reviewing kids' books from a floor photo.

## Routes
- `/books/4f8b2d7c` — secret review link
- `/books-sorter.html` — direct HTML alias
- `/books/demo-photo` — sample demo photo
- `/health` — health check

## Notes
- The app is intentionally separate from HealthOS.
- The HTML UI supports keep/discard marking, category suggestions, duplicate grouping, and export.
